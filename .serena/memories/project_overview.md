# The-Pivot — Project Overview

## Purpose
Agency/portfolio website called "The Pivot" — premium scroll-driven experience with cinematic animations.

## Tech Stack
- **Framework**: Next.js 16.2.4 (App Router, Turbopack)
- **Language**: TypeScript 5
- **Styling**: Tailwind CSS v4 (PostCSS plugin)
- **Animations**: GSAP 3.15 + @gsap/react, ScrollTrigger
- **Smooth Scroll**: Lenis 1.3.23 (synced with GSAP ticker)
- **UI Utilities**: clsx, tailwind-merge, class-variance-authority (CVA)
- **Icons**: lucide-react
- **Base UI**: @base-ui/react

## Design Tokens
- BG: `#0A1510` (dark green)
- CREAM: `#F2F4E7`
- LIME: `#C8D96F`
- Font Display: `var(--font-display)` → Instrument Serif (italic)
- Font Sans: Inter Tight + Plus Jakarta Sans
